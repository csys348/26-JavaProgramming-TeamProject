## Project Title
**O-IoT: Object-Oriented Smart Environment Controller**

---

## Project Description

### English
This project is an **Integrated IoT Management System** developed using Java to demonstrate the core principles of Object-Oriented Programming (OOP). By abstracting various hardware components—such as lighting, thermostats, and security sensors—into a hierarchical class structure, the system allows for centralized monitoring and real-time control. To ensure a seamless demonstration, the project includes a **virtual environment built with mock data** to simulate hardware feedback. Furthermore, a **mobile-optimized UI** will be implemented to allow users to manage all connected devices exclusively via a smartphone interface.

### 한국어
자바를 활용하여 객체지향 프로그래밍(OOP)의 핵심 원칙을 구현한 **통합 IoT 관리 시스템**임. 조명, 온도 조절기, 보안 센서 등 다양한 하드웨어를 계층적 클래스 구조로 추상화하여, 중앙 집중식 모니터링 및 실시간 제어 기능을 제공함. 원활한 시연을 위해 **목 데이터(Mock Data) 기반의 가상 환경**을 구축하여 하드웨어 피드백을 시뮬레이션함. 또한, **모바일 최적화 UI**를 구현하여 사용자가 스마트폰 인터페이스만으로 모든 기기를 조작할 수 있도록 설계함.

## Implementation & Demo Plan
*   **OOP Architecture:** 
    *   `SmartDevice` 추상 클래스 및 인터페이스 기반의 설계.
    *   상속과 다형성을 활용한 기기 확장성 확보.
*   **Mock Environment:** 
    *   물리적 기기 없이도 작동 가능한 가상 센서 및 액추에이터 환경 구성.
    *   실시간 상태 변화를 확인할 수 있는 시뮬레이션 데이터 활용.
*   **Smartphone UI Control:** 
    *   모바일 환경에 최적화된 컨트롤러 UI 구현.
    *   스마트폰 조작을 통한 기기 On/Off, 모드 변경 등 실시간 인터랙션 시연.
